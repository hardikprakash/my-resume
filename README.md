# Hardik Prakash's Resume

LaTeX source files for my professional resume.

## Structure

- `resume.tex` - Main LaTeX document with formatting and package imports
- `content.tex` - Resume content (education, experience, projects, skills)
- `hardiks_resume.pdf` - Compiled PDF output

## Building

To compile the resume, run:

```bash
pdflatex resume.tex
```

The output will be `resume.pdf`.

## Requirements

- LaTeX distribution (TeX Live, MiKTeX, etc.)
- Required packages: latexsym, fullpage, titlesec, marvosym, color, verbatim, enumitem, fancyhdr, xcolor, hyperref
